He gained recognition for his role as Bucky Barnes / Winter Soldier in the Marvel Cinematic Universe media franchise beginning with the film Captain America: The First Avenger (2011), most recently starring in the Disney+ miniseries The Falcon and the Winter Soldier (2021).
