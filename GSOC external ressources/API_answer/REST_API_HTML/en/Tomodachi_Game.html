Tomodachi Game is a Japanese manga series conceptualized by Mikoto Yamaguchi and written and illustrated by Yuki Sato. It has been serialized in Kodansha's Bessatsu Shōnen Magazine since December 2013. A television drama adaptation and two live-action films premiered in 2017. An anime television series adaptation by Okuruto Noboru aired from April to June 2022. A second television drama adaptation titled Tomodachi Game R4 will premiere on TV Asahi in July 2022.
