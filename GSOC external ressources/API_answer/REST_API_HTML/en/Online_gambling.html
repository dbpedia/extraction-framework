\n\nOnline gambling is any kind of gambling conducted on the internet. This includes virtual poker, casinos and sports betting. The first online gambling venue opened to the general public, was ticketing for the Liechtenstein International Lottery in October 1994. Today the market is worth around $40 billion globally each year, according to various estimates.
