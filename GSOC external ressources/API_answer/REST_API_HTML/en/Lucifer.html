Lucifer is one of various figures in folklore associated with the planet Venus. The entity's name was subsequently absorbed into Christianity as a name for the devil. Modern scholarship generally translates the term in the relevant Bible passage, where the Greek Septuagint reads ὁ ἑωσφόρος ὁ πρωὶ, as \"morning star\" or \"shining one\" rather than as a proper noun, Lucifer, as found in the Latin Vulgate.
