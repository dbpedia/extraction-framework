Frank William Abagnale Jr. is an American author and former convicted felon. Abagnale targeted individuals and small businesses yet gained notoriety in the late 1970s by claiming a diverse range of victimless workplace frauds, many of which are now in doubt. In 1980, Abagnale co-wrote his autobiography, Catch Me If You Can, which built a narrative around these claimed victimless frauds. The book later inspired the film of the same name directed by Steven Spielberg in 2002, in which Abagnale was portrayed by actor Leonardo DiCaprio. He has also written four other books. Abagnale runs Abagnale and Associates, a consultancy firm.
