The Irish Republican Army (IRA) is a name used by various paramilitary organisations in Ireland throughout the 20th and 21st centuries. Organisations by this name have been dedicated to irredentism through Irish republicanism, the belief that all of Ireland should be an independent republic free from British rule. They are designated terrorist organisations in Ireland and in the United Kingdom.
