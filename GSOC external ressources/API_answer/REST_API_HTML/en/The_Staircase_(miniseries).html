The Staircase is an American true crime streaming television miniseries created by Antonio Campos, based on the 2004 true crime docuseries of the same name created by Jean-Xavier de Lestrade. The series stars Colin Firth as Michael Peterson, a writer convicted of murdering his wife Kathleen Peterson, who was found dead at the bottom of the staircase in their home. The series premiered on HBO Max on May 5, 2022.
