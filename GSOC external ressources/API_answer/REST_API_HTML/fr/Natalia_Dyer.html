Natalia Dyer [neɪtˈeɪliə ˈdaɪɚ][1], née le 13 janvier 1995[2] à Nashville, est une actrice américaine.

Biographie

Depuis juillet 2016, elle joue dans la série Stranger Things, dans laquelle elle incarne le personnage de Nancy Wheeler, diffusée sur Netflix[3].

Vie privée

Depuis juillet 2017, elle est en couple avec l'acteur Charlie Heaton, rencontré sur le tournage de Stranger Things[4].
