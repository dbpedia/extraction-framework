Antony Starr est un acteur néo-zélandais né le 25 octobre 1975. Il est connu en Nouvelle-Zélande pour avoir incarné les frères jumeaux Jethro et Van West dans la série télévisée à succès Outrageous Fortune (en) (2005-2010). À l'international, sa reconnaissance est liée aux rôles principaux dans deux séries américaines : le shérif Lucas Hood dans Banshee (2013-2016) et le super-héros Le Protecteur dans The Boys (depuis 2019).
