Djibril Glissant est un réalisateur français.

Biographie

Djibril Glissant est un ancien élève de la Fémis (département « Réalisation », promotion 1999)[1], connu surtout pour son long métrage L'Éclaireur sorti en 2006[2], après avoir été présenté notamment au Festival Premiers Plans d'Angers en 2005[3]. Le cinéaste a été le conseiller à la mise en scène d'Emmanuel Mouret pour Laissons Lucie faire. Il est le compagnon de l’actrice Audrey Fleurot[4].
