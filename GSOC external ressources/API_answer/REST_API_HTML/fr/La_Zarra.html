Fatima Zahra HAFDI, dite La Zarra, est une chanteuse canadienne.

Biographie

En 2016, La Zarra se fait connaître par un featuring avec Niro sur Printemps blanc[1],[2].

Lors des NRJ Music Awards 2021, elle est nommée pour la révélation francophone de l'année[3]. Elle y est également présente pour remettre le prix de Groupe/duo international de l'année au groupe Coldplay[4].
