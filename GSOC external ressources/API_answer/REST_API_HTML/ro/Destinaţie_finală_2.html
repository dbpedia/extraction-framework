Destinație finală 2 (engleză: Final Destination 2) este un film de groază din 2003, regizat de David R. Ellis după un scenariu de Eric Bress și J. Mackye Gruber. Este al doilea film din seria de filme Destinație finală. Din distribuția filmului anterior se reîntorc actorii A. J. Cook (închisã într-un ospiciu din proprie inițiativă) și Tony Todd.

Rezumat

Atenție: urmează detalii despre narațiune și/sau deznodământ.

Kimberly Corman (A. J. Cook) are o premoniție prin care reușește să salveze mai multe persoane de un accident în lanț provocat de un autocamion. Împreună formează un grup care încearcă să ocolească moartea.
