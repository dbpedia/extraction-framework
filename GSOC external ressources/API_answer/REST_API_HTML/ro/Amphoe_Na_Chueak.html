Na Chueak (în thailandeză นาเชือก) este un district (Amphoe) din provincia Maha Sarakham, Thailanda, cu o populație de 60.856 de locuitori și o suprafață de 528,198 km².

Componență

Districtul este subdivizat în 10 subdistricte (tambon), care sunt subdivizate în 146 de sate (muban).
