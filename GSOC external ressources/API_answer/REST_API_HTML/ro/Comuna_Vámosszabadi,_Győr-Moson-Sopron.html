Vámosszabadi este un sat în districtul Győr, județul Győr-Moson-Sopron, Ungaria, având o populație de 1.562 de locuitori (2011).

Conform recensământului din 2011, satul Vámosszabadi avea 1.562 de locuitori. Din punct de vedere etnic, majoritatea locuitorilor (97,24%) erau maghiari, cu o minoritate de germani (1,63%). Apartenența etnică nu este cunoscută în cazul a 0,92% din locuitori.[5] Nu există o religie majoritară, locuitorii fiind persoane fără religie (11,14%), luterani (2,94%) și reformați (2,75%). Pentru 82,27% din locuitori nu este cunoscută apartenența confesională.[6]
