Sfântul Onisim (în greacă Ὀνήσιμος, transliterat: Onisimos: n. secolul I d.Hr., Colossae⁠(d), Provincia Denizli, Turcia – d. 68 d.Hr., Roma, Imperiul Roman) a fost un sclav din Imperiul Roman care s-a ridicat până la rangul de patriarh al Constantinopolului, fiind cel de-al treilea care să ocupe această poziție.[1]

Note

  1. ↑ „Onesimus”. Ecumenic Patriarchate of Constantinople. Accesat în 2 d.Hr..
