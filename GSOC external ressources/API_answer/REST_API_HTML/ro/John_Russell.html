John Russell (n. 18 august 1792,[1][2][3][4] Londra Mare, Anglia, Regatul Unit – d. 28 mai 1878,[1][5][2][3] Pembroke Lodge⁠(d), Anglia, Regatul Unit al Marii Britanii și Irlandei[6]) a fost un politician britanic. A fost prim ministru al Marii Britanii între 1846-1852 și 1865-1866.
