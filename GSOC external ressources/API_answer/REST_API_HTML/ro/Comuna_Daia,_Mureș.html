Apold (în maghiară: Apold, în germană: Trappold) este o comună în județul Mureș, Transilvania, România, formată din satele Apold (reședința), Daia, Șaeș și Vulcan.

Conform recensământului efectuat în 2011, populația comunei Apold se ridică la 2.892 de locuitori, în creștere față de recensământul anterior din 2002, când se înregistraseră 2.722 de locuitori.[1] Majoritatea locuitorilor sunt români (70,5%). Principalele minorități sunt cele de romi (21,89%), maghiari (2,8%) și germani (1,42%). Pentru 3,28% din populație, apartenența etnică nu este cunoscută.[2] Din punct de vedere confesional, majoritatea locuitorilor sunt ortodocși (86%), dar există și minorități de penticostali (4,91%), romano-catolici (1,18%), reformați (1,11%) și evanghelici-luterani (1,04%). Pentru 3,28% din populație, nu este cunoscută apartenența confesională.[3]

Politică și administrație

Comuna Apold este administrată de un primar și un consiliu local compus din 13 consilieri. Primarul, Gabriel Mureșan, de la Partidul Național Liberal, este în funcție din octombrie 2020. Începând cu alegerile locale din 2020, consiliul local are următoarea componență pe partide politice:[4]
