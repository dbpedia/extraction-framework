Calamina este un amestec utilizat pentru calmarea pruritului,[1][2] inclusiv cel cauzat de arsuri solare, înțepături de insecte, iederă otrăvitoare (Toxicodendron radicans) sau alte boli ale pielii.[3] De asemenea, poate fi folositoare împotriva iritației pielii uscate.[4] Este aplicată pe piele sub formă de cremă sau loțiune.[1]

Poate induce iritație ca efect secundar.[3] Este un amestec de oxid de zinc și 0,5% oxid feric (Fe2O3).[5] Loțiunile cu calamină mai pot conține și alte ingrediente, precum fenol și hidroxid de calciu.[5][6]

Se află pe lista medicamentelor esențiale ale Organizației Mondiale a Sănătății.[7] Este disponibilă sub formă de medicație generică.
