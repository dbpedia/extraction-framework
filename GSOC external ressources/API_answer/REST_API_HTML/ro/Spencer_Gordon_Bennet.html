Spencer Gordon Bennet (n. 5 ianuarie 1893,[1][2][3][4] Brooklyn, New York City, New York, SUA – d. 8 octombrie 1987,[1][2][3][4] Santa Monica, California, SUA) a fost un producător de film și regizor. Cunoscut drept Regele serialelor, el a regizat mai multe serii de filme decât oricare alt regizor.

Filmografie (selecție)

Debutează ca regizor în 1921 cu filmul Behold the Man, dar primul său film serial este Sunken Silver (1925)
