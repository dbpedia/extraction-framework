Herzl (Hertzl, Herczl) se poate referi la:

  • Theodor Herzl (1860 – 1904), un ziarist și dramaturg evreu
  • Muntele Herzl, un deal situat în vestul Ierusalimului
  • Herțlia, un oraș în Israel
