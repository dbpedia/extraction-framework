Drosophila unipunctata[1] este o specie de muște din genul Drosophila, familia Drosophilidae, descrisă de Patterson în anul 1943.[1][2] Conform Catalogue of Life specia Drosophila unipunctata nu are subspecii cunoscute.[1]
