Dubrovačko primorje este o comună în cantonul Dubrovnik-Neretva, Croația, având o populație de 2.170 de locuitori (2011).

Conform recensământului din 2011, comuna Dubrovačko primorje avea 2.170 de locuitori. Din punct de vedere etnic, majoritatea locuitorilor (98,99%) erau croați. Apartenența etnică nu este cunoscută în cazul a 0,05% din locuitori.[1] Din punct de vedere confesional, majoritatea locuitorilor (98,06%) erau catolici. Pentru 0,65% din locuitori nu este cunoscută apartenența confesională.[2]
