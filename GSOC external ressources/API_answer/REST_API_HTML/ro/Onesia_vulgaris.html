Onesia vulgaris[1] este o specie de muște din genul Onesia, familia Calliphoridae. A fost descrisă pentru prima dată de Robineau-desvoidy în anul 1830.[1][2]

Este endemică în Franța.[1] Conform Catalogue of Life specia Onesia vulgaris nu are subspecii cunoscute.[1]
