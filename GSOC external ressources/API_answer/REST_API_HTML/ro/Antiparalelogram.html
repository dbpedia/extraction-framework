În geometrie, un antiparalelogram[1] este un tip de patrulater autointersectat. La fel ca un paralelogram, un antiparalelogram are două perechi opuse de laturi de lungime egală, iar laturile din perechea mai lungă se intersectează ca într-un mecanism de foarfece.[2][3]
