Bromfeniramina este un antihistaminic derivat de piridină, de generație 1, fiind utilizat în tratamentul alergiilor.[10][11] Printre acestea se numără rinita alergică și urticaria.[10] Calea de administrare disponibilă este cea orală.[10]

Medicamentul a fost descoperit în 1948 și a fost aprobat pentru uz medical în 1955,[12]

Utilizări medicale

Bromfeniramina este utilizată oral în tratamentul reacțiilor alergice (rinită, urticarie, conjunctivită, prurit).[10][11]

Reacții adverse

Fiind un antihistaminic H1 de generație 1, poate produce sedare, somnolență și cefalee.[10] Compusul prezintă activitate anticolinergică intensă, și poate produce xeroftalmie și xerostomie.[13]
