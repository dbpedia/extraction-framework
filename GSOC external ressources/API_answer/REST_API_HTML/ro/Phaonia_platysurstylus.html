Phaonia platysurstylus[1] este o specie de muște din genul Phaonia, familia Muscidae, descrisă de Xue și Wang în anul 2009.[1][2]

Este endemică în Yunnan.[1] Conform Catalogue of Life specia Phaonia platysurstylus nu are subspecii cunoscute.[1]
