Helina fulviventris[1] este o specie de muște din genul Helina, familia Muscidae. A fost descrisă pentru prima dată de Jacques-Marie-Frangile Bigot în anul 1885.[1][2] Conform Catalogue of Life specia Helina fulviventris nu are subspecii cunoscute.[1]
