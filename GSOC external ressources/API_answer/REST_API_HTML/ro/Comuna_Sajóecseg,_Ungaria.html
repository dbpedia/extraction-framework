Sajóecseg este un sat în districtul Miskolc, județul Borsod-Abaúj-Zemplén, Ungaria, având o populație de 1.089 de locuitori (2011).

Conform recensământului din 2011, satul Sajóecseg avea 1.089 de locuitori. Din punct de vedere etnic, majoritatea locuitorilor (99,59%) erau maghiari.[3] Nu există o religie majoritară, locuitorii fiind reformați (40,13%), romano-catolici (28,83%), persoane fără religie (5,23%) și greco-catolici (2,85%). Pentru 22,41% din locuitori nu este cunoscută apartenența confesională.[4]
