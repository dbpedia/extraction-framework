Liolaemus gravenhorstii[5] este o specie de șopârle din genul Liolaemus, familia Tropiduridae, descrisă de Gray 1845.[6][7][1] Conform Catalogue of Life specia Liolaemus gravenhorstii nu are subspecii cunoscute.[6]
