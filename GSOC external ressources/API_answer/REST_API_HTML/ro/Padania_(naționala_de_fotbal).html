Selecționata de fotbal a Padaniei este o selecționată neoficială de fotbal promovată de partidul italian Lega Nord care reprezintă opt regiuni din Italia pe care le numesc Padania. Nu este afiliată la FIFA sau AFC, ci doar la N.F.-Board. A participat la Cupa Mondială Viva în edițiile din 2008, 2009 și 2010, pe care le-a câștigat. Este echipa cu cele mai multe Cupe VIVA câștigate. Și-a stabilit și propria ligă, Lega Federale Calcio Padania.[1]
