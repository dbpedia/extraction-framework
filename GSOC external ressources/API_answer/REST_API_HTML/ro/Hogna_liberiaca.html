Hogna liberiaca[1] este o specie de păianjeni din genul Hogna, familia Lycosidae, descrisă de Roewer, 1959.[2][3]

Este endemică în Liberia.[2] Conform Catalogue of Life specia Hogna liberiaca nu are subspecii cunoscute.[2]
