Kaolinovo (în bulgară Каолиново) este o comună în regiunea Șumen, Bulgaria, formată din orașul Kaolinovo și 15 sate.

La recensământul din 2011, populația comunei Kaolinovo era de 12.093 locuitori. Din punct de vedere etnic, majoritatea locuitorilor (74,12%) erau turci, existând și minorități de romi (13,85%) și bulgari (5,97%).[2][3] Pentru 5,95% din locuitori nu este cunoscută apartenența etnică.
