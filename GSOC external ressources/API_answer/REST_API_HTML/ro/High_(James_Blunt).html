High[1] este o melodie pop-rock scrisă de cântărețul britanic James Blunt și Ricky Ross (solista a trupei scoțiene Deacon Blue), care face parte din albumul Back to Bedlam (2004). Producția a fost realizată de Tom Rothrock și Jimmy Hogarth și a primit o recepție mixtă din partea criticilor de muzică. A fost lansată ca single în august 2004, fără a obține un impact major în topurile Marii Britanii. Cu toate acestea, după succesul melodiei "You're Beautiful" , "High", a fost lansat pentru a doua oară în 2005, ajungând în top-20 la nivel mondial.
