Marius Croitoru (n. 2 octombrie 1980,[1] Giurgiu, Ilfov, România) este un antrenor de fotbal român și jucător retras din activitate care ocupă funcția de antrenor al echipei FC U Craiova 1948.

În perioada 2019-2022, a antrenat echipa FC Botoșani. În iunie 2022, a semnat un contract pentru două sezoane cu FC U Craiova 1948.[2]
