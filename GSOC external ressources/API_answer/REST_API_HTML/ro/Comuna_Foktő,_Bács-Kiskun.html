Foktő este un sat în districtul Kalocsa, județul Bács-Kiskun, Ungaria, având o populație de 1.672 de locuitori (2011).

Conform recensământului din 2011, satul Foktő avea 1.672 de locuitori. Din punct de vedere etnic, majoritatea locuitorilor (96,73%) erau maghiari, cu o minoritate de romi (1,63%). Apartenența etnică nu este cunoscută în cazul a 0,2% din locuitori.[3] Nu există o religie majoritară, locuitorii fiind romano-catolici (47,61%), reformați (18,72%) și persoane fără religie (5,5%). Pentru 27,39% din locuitori nu este cunoscută apartenența confesională.[4]
