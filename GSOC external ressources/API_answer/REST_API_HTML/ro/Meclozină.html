Meclozina (cu denumirea comercială Emetostop),[10] denumită și meclizină, este un antihistaminic H1 derivat de piperazină, de generația 1, fiind utilizat în tratamentul răului de mișcare și vertijului.[11][12] Calea de administrare este orală.[11][10]

Molecula a fost patentată în 1951 și a fost aprobată pentru uz medical în 1953.[13] Este disponibilă sub formă de medicament generic.[14]

Utilizări medicale

Meclozina este utilizată în:[10][11][12]

  • amețeli, greață , vărsături, datorate răului de mișcare (prevenire și tratament)
  • reducerea stării de confuzie, a amețelii și a pierderii echilibrului cauzate de vertij.

Reacții adverse

Fiind un antihistaminic de generația 1, produce somnolență, fatigabilitate și cefalee.[10]
