George Harrison MBE (n. 25 februarie 1943,[1][2][3][4] Liverpool, Anglia, Regatul Unit[5] – d. 29 noiembrie 2001,[1][2][3][4] Los Angeles, California, SUA[6]) a fost un cântăreț, chitarist și compozitor britanic, cel mai tânăr membru al formației The Beatles. George Harrison a fost distins prin includerea sa, ca membru în Ordinul Imperiului Britanic (MBE).
