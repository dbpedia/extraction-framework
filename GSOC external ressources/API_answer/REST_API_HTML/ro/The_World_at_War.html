The World at War (1973–74) este un serial britanic TV documentar care prezintă evenimente din al Doilea Război Mondial. În momentul realizării sale era cel mai scump serial care a fost creat vreodată, cu un buget de 900.000 £.[1] A fost produs de Jeremy Isaacs, narat de Laurence Olivier și cu coloana sonoră creată de Carl Davis. A fost însoțit de o carte, The World at War, care a fost scrisă de Mark Arnold-Forster.
