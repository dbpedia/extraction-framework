Nu confundați cu Trunchi (geometrie).

Trunchiul este un termen anatomic care face referire la partea centrală a organismului majorității animalelor (și la om), aceea parte de unde se extinde gâtul și membrele. [1] Trunchiul include toracele, abdomenul și micul bazin. Este partea cea mai voluminoasă a corpului uman. Legătura dintre trunchi și gât se realizează cu ajutorul unui orificiu, prin care trec traheea, esofagul și vasele sangvine.[2]
