Modifică date / text 

3-pentanolul este un compus organic cu formula chimică C5H12O. Este unul dintre izomerii pentanolului.[2]

Obținere

3-pentanolul poate fi obținut în urma reacției de hidroliză a 3-cloropentanului, hidratării 2-pentenei sau hidrogenării 3-pentanonei.[3]
