FC Academia Chișinău este un club de fotbal din Republica Moldova, care joacă în orașul-capitală Chișinău. Culorile clubului sunt alb-albastru. A fost înființat în 2006 de Igor Dobrovolski sub denumirea "FC Academia UTM Chișinău" pe care și-a schimbat-o în "FC Academia Chișinău" {debarasându-se de sintagma "UTM"} în data de 25 iunie 2012[1][2].
