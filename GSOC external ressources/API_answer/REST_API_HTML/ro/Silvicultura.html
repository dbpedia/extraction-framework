Silvicultura este disciplina care se ocupă cu studiul pădurilor și plantațiilor de arbori, și cu gospodărirea lor.

Primele publicații ale domeniului

  • Revista Montes (Revista forestieră, Spania) - 1868[1]
  • Šumarski list (Revista forestieră, Croația) - 1877 [2]
  • Revista pădurilor (România, 1881–1882; 1886–prezent),[3] cea mai veche revistă din România
