Várgesztes (în germană Gestitz) este un sat în districtul Tatabánya, județul Komárom-Esztergom, Ungaria, având o populație de 572 de locuitori (2011).

Conform recensământului din 2011, satul Várgesztes avea 572 de locuitori. Din punct de vedere etnic, majoritatea locuitorilor (69,29%) erau maghiari, cu o minoritate de germani (30,71%).[3] Din punct de vedere confesional, majoritatea locuitorilor (52,97%) erau romano-catolici, existând și minorități de persoane fără religie (6,82%) și reformați (3,67%). Pentru 35,14% din locuitori nu este cunoscută apartenența confesională.[4]
