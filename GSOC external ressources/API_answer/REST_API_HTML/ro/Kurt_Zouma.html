Kurt Zouma (n. 27 octombrie 1994,[1][2] Lyon, Ron-Alpi, Franța) este un fotbalist francez care în prezent evoluează la clubul de fotbal West Ham din Premier League și la echipa națională de fotbal a Franței pe postul de fundaș sau mijlocaș.
