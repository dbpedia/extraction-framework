Rhampholeon acuminatus[1] este o specie de cameleoni din genul Rhampholeon, familia Chamaeleonidae, descrisă de Jean Mariaux și Colin Tilbury în anul 2006.[2][3] Conform Catalogue of Life specia Rhampholeon acuminatus nu are subspecii cunoscute.[2]
