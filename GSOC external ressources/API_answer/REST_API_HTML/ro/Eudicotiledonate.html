Eudicotiledonate (Eudicotyledoneae sau Eudicotidae) este o cladă monofiletică a plantelor angiosperme care a fost denumită tricolpates sau dicote non-magnoliide de către autorii anteriori. Termenii botanici au fost introduși în 1991 de către botanistul evoluționar James A. Doyle și paleobotanistul Carol L. Hotton pentru a accentua diferențele evoluționare ulterioare a dicotiledonatelor tricolpate de dicotiledonatele timpurii, mai puțin specializate.[1]
