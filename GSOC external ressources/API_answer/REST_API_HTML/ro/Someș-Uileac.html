Someș-Uileac, mai demult Uileacu Hododului, (în maghiară Szilágyújlak, alternativ Szamosújlak, colocvial Újlak, în germană Neudorf, în trad. "Satu Nou"), este un sat ce aparține orașului Ulmeni din județul Maramureș, Transilvania, România.

Istoric

Prima atestare documentară: 1383 (Ujlak). [1]

Etimologie

Etimologia numelui localității: din hidron. Someș (cuvânt autohton, cf. Samus) + Uileac (< magh. uj "nou" + magh. lak „locuință, domiciliu"). [2]

Demografie

La recensământul din 2011, populația era de 787 locuitori. [3]
