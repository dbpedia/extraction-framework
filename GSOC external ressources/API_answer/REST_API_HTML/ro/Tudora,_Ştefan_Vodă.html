Tudora este o comună în raionul Ștefan Vodă din Republica Moldova, situată pe malul drept al fluviului Nistru.

Geografie

În partea de nord a satului este amplasată Râpa de Piatră, o arie protejată din categoria monumentelor naturii de tip geologic sau paleontologic.[2]
