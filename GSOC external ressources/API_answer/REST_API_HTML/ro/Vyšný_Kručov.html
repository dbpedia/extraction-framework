Vyšný Kručov este o comună slovacă, aflată în districtul Bardejov din regiunea Prešov. Localitatea se află la altitudinea de 200 m deasupra n.m., se întinde pe o suprafață de 3,96 km2 și în 2017 număra 149 de locuitori.[1]

Istoric

Localitatea Vyšný Kručov este atestată documentar din 1438.
