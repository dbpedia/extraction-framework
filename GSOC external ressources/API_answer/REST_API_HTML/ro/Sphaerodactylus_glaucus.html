Sphaerodactylus glaucus[3] este o specie de șopârle din genul Sphaerodactylus, familia Gekkonidae, descrisă de Cope 1866.[4][5] Conform Catalogue of Life specia Sphaerodactylus glaucus nu are subspecii cunoscute.[4]
