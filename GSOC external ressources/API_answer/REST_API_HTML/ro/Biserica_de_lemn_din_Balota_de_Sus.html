Biserica de lemn din Balota de Sus se află în localitatea omonimă din județul Dolj, poartă hramul „Sfinții Voievozi” și e datată din anul 1812, pictată în 1858. Biserica este înscrisă pe noua listă a monumentelor istorice, LMI 2004: DJ-II-m-B-08184i
