Les tortillards (Trenurile de navetiști) este un film franțuzesc de comedie din 1960. Este regizat de Jean Bastia după un scenariu de Pierre Gaspard-Huit. Louis de Funès interpretează rolul industriașului Émile Durand, care nu dorește ca fiul său Gérard (interpretat de Roger Pierre) să plece cu Suzy Beauminet (Danièle Lebrun), o fată de la un teatru ambulant. Filmul a fost cunoscut ca "Io... mio figlio e la fidanzata" în Italia și ca "Der Umstandskrämer" în Germania de Est.[1]
