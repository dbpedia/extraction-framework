Pitomača este o comună în cantonul Virovitica-Podravina, Croația, având o populație de 10.059 de locuitori (2011).

Conform recensământului din 2011, comuna Pitomača avea 10.059 de locuitori. Din punct de vedere etnic, majoritatea locuitorilor (98,62%) erau croați. Apartenența etnică nu este cunoscută în cazul a 0,05% din locuitori.[1] Din punct de vedere confesional, majoritatea locuitorilor (97,31%) erau catolici. Pentru 0,63% din locuitori nu este cunoscută apartenența confesională.[2]
